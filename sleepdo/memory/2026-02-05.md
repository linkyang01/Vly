# 2026-02-05 - SleepDo MVP 设计讨论

## 核心定位确定

- **目标用户**: 失眠/难入睡人群
- **核心差异化**: 4-7-8 呼吸（哈佛医学院认证）
- ** slogan 方向**: "多种助眠方法，找到适合你的"

## MVP 功能边界

### MVP 功能
| 功能 | 状态 | 备注 |
|------|------|------|
| 5种助眠方法 | ✅ | 4-7-8呼吸、渐进式肌肉放松、身体扫描、2-1-6呼吸、白噪音 |
| Watch 心率/HRV 实时显示 | ✅ | 数据采集 + 展示 |
| 睡眠状态指示 | ✅ | 简化版 |
| 渐强唤醒 | ✅ | 原生震动 |
| AI 浅睡检测 | ❌ 暂缓 | 需要用户数据积累 |

### 差异化分析
- **Calm/Headspace**: 冥想 ≠ 4-7-8呼吸
- **Sleep Cycle**: 追踪 ≠ 呼吸引导
- **Noisli**: 混音 ≠ 呼吸引导
- **小睡眠**: 免费声音 ≠ 科学方法论

**结论**: 4-7-8呼吸是 SleepDo 核心护城河，市面唯一认真做的。

## Apple Watch 策略

### MVP Watch 功能
| 功能 | 复杂度 | 说明 |
|------|--------|------|
| 实时心率/HRV 显示 | 低 | 每秒/每分钟更新 |
| 睡眠状态指示 | 低 | 简化判断 |

### MVP 不做
- 渐强唤醒（后续）
- 自动睡眠检测（后续）
- 智能唤醒算法（需要数据积累）

### 理由
- MVP 关键是快速验证核心假设
- 没有足够数据时算法准确率不高
- 用户体验优先

## 商业模式

### 双区策略

| 版本 | 语言 | 定价 | 市场 |
|------|------|------|------|
| SleepDo (International) | 英文 | $4.99/月 | 美区/全球 |
| 木木眠 (中国版) | 简体中文 | ¥12/月 | 中国区 |

### 理由
- **美区先上**: 验证付费意愿，无版号问题
- **中国区后上**: 需申请版号（1-2个月）
- **定价差异**: 符合各区消费力

### 定价理由
- **美区 $4.99/月**: 低门槛尝试（对比 Calm $70/年）
- **中国区 ¥12/月**: 比免费竞品有差异化（对比小睡眠）

### 免费功能（转化漏斗）
- 5种助眠方法全部可用
- 基础心率显示
- 3天睡眠记录

### 付费解锁（7天后）
- 完整心率趋势
- 15个高级声音
- 30天数据存储

## 用户转化漏斗

```
Day 1: 下载 → 选择方法 → 使用 → "有效/无效"
Day 2-3: 展示效果数据 → "比昨天快了5分钟"
Day 4-5: 引导升级 → "解锁完整趋势"
Day 7: 试用结束 → 弹出订阅
Day 8: 付费 / 流失
```

### 关键指标
- Day 1 完成首次助眠: >60%
- Day 3 看到效果数据: >40%
- Day 7 看到7天趋势: >20%
- 7天付费转化率: >5%
