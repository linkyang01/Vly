# SleepDo MVP 实施计划

> 创建日期: 2026-02-05

## MVP 目标

验证核心假设：**声音 + 4-7-8呼吸 能帮助失眠用户入睡**

### 成功指标
- 7天留存率 > 30%
- 试用 → 付费转化率 > 5%
- 用户入睡时间平均缩短 > 20%

---

## MVP 功能清单

### 核心功能 (必须实现)

| 功能 | 描述 | 工期 |
|------|------|------|
| 声音播放 | 10个基础声音，单个播放 | 3天 |
| 4-7-8呼吸 | 引导动画 + 语音 | 3天 |
| 手动闹钟 | 渐强音量 | 2天 |
| 睡眠记录 | 手动记录入睡/起床时间 | 2天 |
| 订阅墙 | 7天试用后弹出 | 2天 |

### 用户流程

```
Day 1: 下载 → 引导体验4-7-8呼吸 → 选择声音入睡
Day 2-7: 记录睡眠 → 看到进步
Day 8: 订阅墙 → 付费 / 不付费
```

### 后续功能 (MVP成功后添加)

| 功能 | 触发条件 |
|------|---------|
| Apple Watch 集成 | MVP 成功 + 1000+ 用户 |
| 成就系统 | 用户要求 |
| 智能唤醒 | 有 Watch 数据后 |
| 混音器 | 用户要求 |
