# 睡前仪式设计文档

> 最后更新: 2026-02-05

## 功能概览

睡前仪式是帮助用户入睡前的放松流程，包含：

| 功能 | 图标 | 时长 |
|------|------|------|
| 4-7-8 呼吸练习 | 💨 | 3分钟 |
| 冥想练习 | 🧘 | 5分钟 |
| 助眠声音 | 🔊 | 可选 |

## 4-7-8 呼吸练习

### 设计规格
- **总时长**: 3分钟
- **循环周期**: 吸气4秒 → 屏息7秒 → 呼气8秒
- **动画**: 圆形收缩扩张

### UI 元素
```
    ○ 吸气 (4s)
   ◐
  ◑
 ◐
○ 呼气 (8s)
```

### 倒计时显示
- 大字体数字居中
- 当前阶段文字提示
- 进度条显示剩余时间

### 音频引导
- 轻柔的语音引导呼吸节奏
- 背景音乐（可选）

## 冥想练习

### 三种模式

#### 1. 身体扫描 (Body Scan)
- 引导注意力从头到脚扫描身体
- 识别并放松紧张部位
- **时长**: 5分钟

#### 2. 感恩练习 (Gratitude)
- 回忆今天值得感恩的3件事
- 培养积极心态
- **时长**: 5分钟

#### 3. 呼吸冥想 (Breath Meditation)
- 专注于呼吸节奏
- 计数呼吸次数
- **时长**: 5分钟

### UI 结构

```
├─────────────────────────┤
│                         │
│    "今天有什么好事发生？"  │
│                         │
│  • 与朋友聊天           │
│  • 完成工作任务         │
│  • 吃到好吃的食物       │
│                         │
├─────────────────────────┤
│    [开始]  [跳过]       │
└─────────────────────────┘
```

## 助眠声音

### 功能
- 在仪式期间播放助眠声音
- 可混合多个声音
- 可设置定时关闭

### 支持的声音
复用 SoundsView 中的 42 个声音，重点：
- 雨声 (rain)
- 森林 (forest)
- 海浪 (waves)
- 壁炉 (fire)
- 白噪声 (white noise)

### 混音功能
- 独立音量控制
- 预设组合

## 集成到睡眠首页

### 卡片设计
```
┌─────────────────────────────────┐
│ 💤 睡前仪式                      │
├─────────────────────────────────┤
│  💨 呼吸练习    3分钟            │
│  🧘 冥想练习    5分钟            │
│  🔊 助眠声音    可选             │
├─────────────────────────────────┤
│      [开始仪式]                  │
└─────────────────────────────────┘
```

### 触发方式
1. 用户手动点击
2. 睡前提醒通知
3. 设置中的"自动开启睡前仪式"

## 数据模型

```swift
struct BedtimeRoutine {
    let id: UUID
    var breathingEnabled: Bool
    var breathingDuration: Int  // 分钟
    var meditationEnabled: Bool

enum MeditationType {
    case bodyScan
    case gratitude
    case breathMeditation
}
```
