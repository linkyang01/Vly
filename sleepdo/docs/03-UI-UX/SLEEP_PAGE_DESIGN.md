# 睡眠页面设计

> 最后更新: 2026-02-05

## 页面结构

### 顶部区域
- 睡眠状态卡片（当前是否在睡眠中）
- 睡眠目标显示
- 快捷操作按钮

### 中间区域
- 睡眠数据概览
- 最近睡眠记录
- 成就进度

### 底部区域
- 睡前仪式入口
- 声音播放控制

## 设计规范

### 颜色
- 背景: 紫色渐变 #0D0D1A → #1A1A3E → #2D1B4E
- 强调色: #8B5CF6 (紫色)
- 文字: 白色
- 副文本: 灰色

### 卡片样式
- 圆角: 20pt
- 背景: Color(red: 0.1, green: 0.1, blue: 0.15).opacity(0.8)
- 内边距: 20pt

## 组件

### SleepStatusCard
显示当前睡眠状态

### SleepGoalCard
显示睡眠目标进度

### BedtimeRoutineCard
睡前仪式入口

### QuickActionRow
快捷操作按钮行
