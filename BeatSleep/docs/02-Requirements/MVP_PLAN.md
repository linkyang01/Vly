# BeatSleep MVP 计划

> 版本: 1.0.0  
> 更新日期: 2026-02-06  
> 状态: 正式发布

---

## MVP 目标

验证核心假设：**4-7-8 呼吸法 + Watch 数据展示能帮助失眠用户快速入睡。**

### 成功指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 7天付费转化率 | >5% | 付费意愿 |
| 7天留存率 | >30% | 用户粘性 |
| DAU/下载率 | >30% | 日活率 |

---

## MVP 功能范围

### 核心功能（必须实现）

| 功能 | 描述 | 优先级 | 工时 |
|------|------|--------|------|
| 4-7-8 呼吸 | 核心差异化，动画+语音 | P0 | 3天 |
| 渐进式肌肉放松 | 第二选择 | P1 | 2天 |
| 身体扫描 | 第三选择 | P1 | 2天 |
| "2-1-6" 呼吸 | 入门/降级 | P2 | 1天 |
| 白噪音 | 环境辅助 | P2 | 1天 |
| Watch 心率/HRV | 数据展示 | P1 | 2天 |
| 睡眠记录 | 效果追踪 | P1 | 1天 |
| 设置页 | 基本配置 | P2 | 1天 |

### MVP 不做（后续版本）

| 功能 | 原因 |
|------|------|
| AI 浅睡检测 | 需要数据积累 |
| 智能唤醒算法 | 需要数据积累 |
| 自动睡眠检测 | MVP 简化 |
| 成就系统 | 后续迭代 |
| 高级声音包 | 订阅功能 |

---

## 用户流程

```
Day 1: 下载 → 引导选择助眠方法 → 使用 → "哈佛认证方法"
    │
Day 3: 效果展示 → "你昨晚心率72→65" / "HRV改善8%"
    │
Day 5: 引导升级 → "解锁完整趋势图 + 高级声音"
    │
Day 7: 订阅墙 → 试用结束
    │
Day 8: 付费 / 流失
```

| 风险 | 可能性 | 影响 | 缓解措施 |

---

## 成功关键

1. **4-7-8 呼吸法是核心差异化** - 市面唯一认真做的
2. **多种方法让用户找到适合自己的** - 降低流失率
3. **Watch 数据直观展示效果** - 差异化竞争点
4. **7天内让用户看到改善** - 转化关键
