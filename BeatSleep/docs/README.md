## 核心定位

| 项目 | 值 |
|------|------|
| App 名称 | BeatSleep |
| 副标题 | Fall Asleep Faster |
| 核心价值 | 帮助快速入睡 |
| 目标用户 | 失眠/难入睡人群 |
| 核心差异化 | 4-7-8 呼吸法（哈佛认证） |
