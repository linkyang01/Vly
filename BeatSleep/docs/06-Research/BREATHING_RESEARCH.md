## 生理机制

### 4-7-8 原理

```
吸气 (4s) → 增加氧气 → 准备阶段
    ↓
屏息 (7s) → 氧气交换 → 肺部充盈
    ↓
呼气 (8s) → 激活副交感 → 放松反应
    ↓
重复 → 心率下降 → 入睡准备
```

### 效果指标

| 指标 | 变化 |
|------|------|
| 心率 | -5 ~ -10 BPM |
| 皮质醇 | -15% |
| 焦虑感 | -40% |
| 入睡时间 | -50% |
