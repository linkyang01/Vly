## [1.0.0] - 2026-02-06

### Added

- Initial project creation
- Core app structure with XcodeGen
- 5 breathing techniques data models
- Sleep session tracking
- Apple Watch app skeleton
- CI/CD configuration
- Backup scripts
- Code standards documentation

### Features

- SleepHomeView with today's card and history
- TechniquesView with 5 method selection
- BreathingSessionView with animation (4-7-8)
- WatchDataView with heart rate/HRV display
- SettingsView with subscription placeholder
- Watch app with 3 pages

### Documentation

- CODING_STANDARDS.md - Code style guidelines
- PROJECT_STRUCTURE.md - Complete project structure
- docs/01-Architecture/README.md - Architecture design
- docs/02-Requirements/MVP_PLAN.md - MVP plan
- docs/04-Data-Models/DATA_MODELS.md - Data models
- docs/README.md - Documentation index

### Infrastructure

- GitHub Actions CI/CD workflow
- XcodeGen project configuration
- Backup scripts (full, quick, sync, restore)
