## 1. 产品概述

### 1.1 产品定位

**一句话描述**：
> 最美的 iOS 喝水追踪工具，无广告、买断制、原生 Apple Watch 体验。

### 1.2 核心价值主张

| 维度 | 承诺 |
|------|------|
| **美观** | iOS 26 液态玻璃 UI，最美喝水 App |
| **简单** | 3 秒完成每日打卡 |
| **联动** | Apple Watch 原生深度集成 |
| **诚实** | 无广告、买断制、无隐形消费 |

### 1.3 成功指标

- **下载量**：首月 10,000 下载
- **7 日留存**：> 40%
- **付费转化**：> 8%（对比行业 3-5%）
- **App Store 评分**：4.8 分+

## 2. 目标用户

### 2.1 核心用户画像

| 属性 | 描述 |
|------|------|
| **年龄** | 25-45 岁 |
| **设备** | iPhone + Apple Watch 用户 |
| **场景** | 办公室、健身人群、健康意识强 |
| **痛点** | 忙到忘记喝水、现有 App 太丑/广告太多 |

### 2.2 用户故事

```
作为 [办公室白领]
我想 [快速记录每日喝水]
以便 [保持健康、提升精力]

作为 [健身爱好者]
我想 [精确追踪每日饮水量]
以便 [达成训练目标、辅助增肌/减脂]

作为 [Apple Watch 用户]
我想 [抬腕快速打卡]
以便 [不掏出手机、保持运动节奏]
```

## 3. 核心功能

### 3.1 MVP 功能（必须实现）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **快速喝水** | 一键添加 200ml/250ml/500ml | P0 |
| **今日目标** | 默认 2000ml，可自定义 | P0 |
| **今日进度** | 环形进度条可视化 | P0 |
| **Widget 小组件** | 锁屏/桌面快速打卡 | P0 |
| **Apple Watch 打卡** | 抬腕添加饮水记录 | P0 |
| **历史记录** | 查看过去 7 天/30 天数据 | P1 |
| **推送提醒** | 每隔 1-2 小时提醒喝水 | P1 |
| **成就系统** | 连续打卡解锁徽章 | P2 |

### 3.2 高级功能（后续迭代）

| 功能 | 描述 | 触发条件 |
|------|------|---------|
| **饮水趋势** | 周/月维度可视化图表 | MVP 成功后 |
| **自定义提醒** | 设置具体时间段的提醒 | 用户要求 |
| **家庭共享** | 追踪家人饮水数据 | 10,000+ 用户 |
| **Apple Health 同步** | 写入每日饮水量到 HealthKit | 用户要求 |

## 5. Apple Watch 集成

### 5.1 Watch App 功能

| 功能 | 描述 |

```
iPhone App  ←→  Watch App
     ↑              ↑
     └────→ HealthKit ←─┘
```

### 5.3 关键场景

```
场景：用户健身时
─────────────────────
1. 运动中感到口渴
2. 抬起 Apple Watch
3. 点击 Complication
4. 选择 250ml
5. 震动反馈 + 进度更新
6. 无需掏出手机
```

### 6.3 核心数据模型

```swift
// 饮水记录
struct WaterRecord: Identifiable, Codable {
    let id: UUID
    let amount: Int      // ml
    let timestamp: Date
    let source: DataSource  // .iphone, .watch, .widget
}

// 用户设置
struct UserSettings: Codable {
    var dailyGoal: Int        // 默认 2000ml
    var reminderEnabled: Bool
    var reminderStartHour: Int   // 8
    var reminderEndHour: Int     // 22
    var unit: MeasurementUnit    // .ml, .oz
    var watchSyncEnabled: Bool
}

// 数据源
enum DataSource: String, Codable {
    case iphone
    case watch
    case widget
}
```
