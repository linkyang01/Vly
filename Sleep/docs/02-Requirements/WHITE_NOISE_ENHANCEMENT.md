# 白噪音功能增强（v2.0）

> **版本**: 2.0  
> **状态**: 计划中  
> **优先级**: 中

---

## 概述

扩展白噪音功能，从单一类型扩展为多种声音类型 + 组合混音功能。

---

## 功能规划

### 1. 多声音类型

| 类型 | 图标 | 描述 | 失眠效果 |
|------|------|------|---------|
| 🌧️ 雨声 | `cloud.rain` | 柔和的雨声，掩盖突发噪音 | ⭐⭐⭐⭐⭐ |
| 🌊 海浪 | `wave.3` | 节奏感的海浪声 | ⭐⭐⭐⭐⭐ |
| 🌲 风声 | `wind` | 轻柔的风声 | ⭐⭐⭐⭐ |
| 🔥 壁炉 | `fire` | 温暖的壁炉声 | ⭐⭐⭐⭐ |
| 🌲 森林 | `leaf` | 自然的森林氛围 | ⭐⭐⭐⭐ |
| 💧 河流 | `drop` | 平静的流水声 | ⭐⭐⭐⭐ |

### 2. 组合混音功能

#### 预设组合
- **雨中海浪**：雨声 70% + 海浪 50%
- **自然之声**：雨声 50% + 森林 40% + 风声 30%
- **温暖壁炉**：壁炉 60% + 雨声 40%

#### 自定义混音
- 每种声音独立音量滑块（0-100%）
- 独立开关（开启/关闭）
- 总体音量控制
- 保存自定义组合

## 技术实现

### 数据模型
```swift
enum WhiteNoiseType: CaseIterable {
    case rain, ocean, wind, fire, forest, river
}

struct SoundMixItem {
    var type: WhiteNoiseType
    var volume: Double  // 0.0 - 1.0
    var isEnabled: Bool
}

struct SoundMixPreset {
    var name: String
    var items: [SoundMixItem]
}
```

### 服务层
- `WhiteNoisePlayer` - 管理多个 AVAudioPlayer
- 支持同时播放多种声音
- 独立音量控制

### 资源需求
- 需要准备 6 种音频文件（mp3）
  - `rain.mp3`
  - `ocean.mp3`
  - `wind.mp3`
  - `fire.mp3`
  - `forest.mp3`
  - `river.mp3`
