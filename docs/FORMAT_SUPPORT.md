# Vly 格式支持清单

## 1. 目标

Vly 的长期目标是成为全能媒体播放器，但格式支持必须分阶段验证，不能只停留在宣传描述。

本文件用于定义：

- 必须支持的封装格式；
- 必须验证的视频编码；
- 必须验证的音频编码；
- 字幕支持范围；
- 第一阶段测试清单。

## 2. 第一阶段必须验证

| 类型 | 格式 |
|---|---|
| 视频封装 | MP4、MKV |
| 视频编码 | H.264、H.265 / HEVC |
| 音频编码 | AAC、AC3 |
| 字幕 | 外挂 SRT 暂列为阶段 2，第一阶段只记录是否存在 |

第一阶段只要求 MP4 与 MKV 稳定播放，不要求覆盖所有格式。

## 3. 第二阶段重点验证

| 类型 | 格式 |
|---|---|
| 视频封装 | AVI、MOV、FLV、TS、M2TS |
| 视频编码 | MPEG-4、VP9、AV1 |
| 音频编码 | DTS、FLAC、MP3、Opus |
| 字幕 | SRT、ASS、SSA、内嵌字幕 |

## 5. 测试矩阵

每个格式测试时记录：

- 文件名；
- 封装格式；
- 视频编码；
- 音频编码；
- 字幕类型；
- 是否能打开；
- 是否能播放；
- 是否能拖动进度；
- 是否有音频；
- 是否卡顿；
- 错误信息。

## 6. 第一阶段验收

第一阶段只要满足以下条件即可进入下一阶段：

- MP4 / H.264 / AAC 正常播放；
- MP4 / H.265 / AAC 正常播放；
- MKV / H.264 / AAC 或 AC3 正常播放；
- MKV / H.265 / AAC 或 AC3 正常播放；
- 播放、暂停、进度、音量可用。
