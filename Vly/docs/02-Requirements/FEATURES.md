# Vly 功能需求文档

> 版本: 1.0
> 更新日期: 2026-02-06

## 1. 概述

Vly 是一个简洁优雅的 macOS/iOS 视频播放器，目标是提供流畅的本地和网络视频播放体验。

| 格式 | 支持情况 | 备注 |

## 5. 非功能需求

### 5.1 性能需求

| 指标 | 要求 |
|------|------|
| 启动时间 | < 2秒 |
| 视频加载时间 | < 1秒 (本地) |
| 内存占用 | < 100MB (1080p) |
| CPU 占用 | < 30% (正常播放) |

### 5.2 兼容性

| 平台 | 最低版本 |
|------|----------|
| macOS | 12.0 Monterey |
| iOS | 15.0 |
| Xcode | 15.0 |
