# 快速备忘录

## 使用方式

**一行命令**：
```bash
echo "备忘录内容" | memo notes -a -f "Notes"
```

## 集成到 OpenClaw

把 skills/memo-memos/ 添加到 OpenClaw 的 skills 目录即可使用。

## 示例

```
echo "OpenClaw 重装恢复步骤" | memo notes -a -f "Notes"
```
