## 设计规范

### 颜色主题

| 元素 | 颜色 | 说明 |
|------|------|------|
| **主色调** | `#22C55E` | 绿色 - 护眼、健康 |
| **渐变背景** | `#ECFDF5` → `#D1FAE5` → `#A7F3D0` | 浅绿渐变 |
| **强调色** | `#10B981` | 成功、达成 |
| **背景** | `Color.clear` | 透明（显示系统壁纸） |
| **文字** | `#1F2937` / `#6B7280` | 深灰/中灰 |
| **卡片背景** | `Color(UIColor.systemBackground).opacity(0.9)` | 半透明卡片 |

### 字体

| 元素 | 字体 | 大小 |
|------|------|------|
| 大标题 | SF Pro Display Bold | 34 |
| 标题 | SF Pro Display Bold | 24 |
| 副标题 | SF Pro Display Medium | 18 |
| 正文 | SF Pro Text Regular | 16 |
| 小字 | SF Pro Text Regular | 14 |
| 标签 | SF Pro Text Medium | 12 |

## 设计原则

### 1. 一致性

- 与 BeatSleep 使用相同的设计模式
- 颜色主题与护眼概念一致（绿色）
- 保持界面简洁、优雅

### 2. 轻量化

- 不打扰用户
- 操作简单直接
- 20 秒内完成护眼

### 3. 数据可视化

- 统计图表清晰
- 成就展示直观
- 进度一目了然

### 4. 激励性

- 连续天数突出显示
- 成就解锁有仪式感
- 友好的鼓励文案
