## 眼疲劳科学基础

### 20-20-20 规则

**来源**: American Optometric Association (美国验光协会)

**规则**: 每使用电子设备 **20 分钟**，看 **20 英尺（约 6 米）** 远的地方，至少 **20 秒**

### 数字眼疲劳 (Digital Eye Strain / Computer Vision Syndrome)

| 症状 | 发生率 |
|------|--------|
| 眼干 | ~50% |
| 眼疲劳 | ~65% |
| 头痛 | ~30% |
| 视力模糊 | ~25% |

### 蓝光影响

| 类型 | 波长 | 影响 |
|------|------|------|
| 蓝光 | 400-500nm | 影响褪黑素分泌，抑制睡眠 |

## 核心功能设计

### 基础功能

| 功能 | 描述 | 技术实现 |
|------|------|---------|
| **定时提醒** | 每 20 分钟提醒休息 | UserNotifications |
| **用眼统计** | 今日/本周用眼时长 | Screen Time API |
| **20-20-20 引导** | 引导看远处 20 秒 | 动画 + 倒计时 |
| **护眼模式** | 屏幕色调调节 | Night Shift 联动 |

### 进阶功能

| 功能 | 描述 | 技术实现 |
|------|------|---------|
| **Apple Watch 联动** | 手表震动提醒 | WatchConnectivity |
| **用眼热力图** | 一天中疲劳时段分析 | Charts + 数据聚合 |
| **姿势提醒** | 检测用眼距离（需摄像头）| ARKit / 红外传感器 |
| **眨眼提醒** | 提醒多眨眼防止眼干 | - |

### EyeCare 定价建议

#### 方案 A：参考 BeatSleep（推荐）

| 项目 | 价格 | 说明 |
|------|------|------|
| 免费版 | ✅ | 核心功能全部免费 |
| 终身高级版 | ¥49.99 | 一劳永逸 |

**理由**：
- 竞品多数靠订阅赚钱 → 用户怕被"割韭菜"
- 一次性付费更友好、口碑更好
- 和 BeatSleep 定价一致 → 用户认知统一
- 核心功能免费 → 降低使用门槛

#### 方案 B：免费 + 订阅

| 项目 | 价格 | 说明 |
|------|------|------|
| 免费版 | ✅ | 基础功能 |
| Pro 月订阅 | ¥8/月 | 高级统计 |
| Pro 年订阅 | ¥68/年 | 便宜 30% |

**理由**：
- 持续收入
- 可以持续迭代功能
- 风险：用户怕自动续费

---

### 建议方案：方案 A（免费 + 终身制）

**原因**：

| 优势 | 说明 |
|------|------|
| **用户友好** | 一次付费，永久使用 |

| **和 BeatSleep 一致** | 定价策略统一 |

### 和 BeatSleep 协同

| 维度 | 协同方式 |
|------|---------|
| **用户交叉** | BeatSleep 用户 ⭐⭐⭐ 推荐 EyeCare |

### 风险与对策

| 风险 | 对策 |
|------|------|
| 用户觉得"就一个提醒功能，不值 ¥49" | ✅ 免费版功能足够实用 |
| 竞品打价格战 | ✅ 差异化在质量和生态，不在价格 |
| 系统功能替代（iOS 越来越强） | ✅ 专注引导体验和助眠联动 |

---

### 结论

**推荐策略**：

1. **免费版**：核心功能全开，建立用户基础
2. **终身制 ¥49.99**：一次性付费，无订阅压力
3. **差异化**：极简体验 + BeatSleep 联动
4. **长期目标**：形成"睡眠 + 护眼"健康闭环

## 设计原则

1. **轻量级** - 不打扰用户工作
2. **温柔提醒** - 不是强制中断
3. **数据可视化** - 让用户看到进步
4. **隐私保护** - 本地存储用眼数据
